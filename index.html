Welcome to Chrome Tools!

How do I access it?

You may already be familiar with Chrome Dev Tools. In case you aren't, you can access those tools several ways.

  • Through the Chrome menu. More Tools --> Developer Tools
  • Right-click and select inspect (Elements tab)
  • Keyboard shortcut Alt-Cmd-I (Elements tab)
  • Keyboard shortcut Alt-Cmd-J (Console tab)

Tabs? Huh?

Now that you have the Chrome tools open, notice the tabs listed along the top.

  • Elements: Shows you the HTML. Hovering on a tag shows you it's box model properties.
  • Console: Anything JavaScript outputs (including errors) goes here
  • Sources: This shows you all the files that went into this webpage. Debugging happens here.
  • Network: This tab shows you stuff about how the files on the page loaded including status codes, request and response headers, etc.
  • The rest we won't be using much, but you can learn about them on your own!

What's that phone icon thingy?

To the left of your tabs you will see a little icon that looks like a cell phone next to a tablet. Click on it and see what it does!
